## Опис
Програма реалізує систему управління студентами з використанням масивів у JavaScript.

## Функціональні можливості
- Додавання студентів
- Фільтрація за віком, балом та факультетом
- Сортування за ім'ям, віком та балом
- Пошук студента за ім'ям
- Обчислення статистики:
  - середній бал
  - найкращий студент
  - розподіл за факультетами

## Використані методи масивів
- filter()
- sort()
- find()
- reduce()

## Технології
- JavaScript (ES6+)
- Node.js

## Запуск
node index.js

## Приклад структури даних
{
  id: 1,
  name: "Іван Петренко",
  age: 20,
  grade: 85.5,
  faculty: "IT"
}
